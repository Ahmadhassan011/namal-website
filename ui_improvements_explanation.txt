1. CARD DESIGN IMPROVEMENTS
================================================================================

### What Was Changed:

1.1 VISUAL STYLE ENHANCEMENTS
-----------------------------
- Increased border-radius from 12px to 16px for a softer, more modern look
- Added subtle gradient backgrounds on hover
- Enhanced shadows for depth perception (from shadow-sm to shadow-lg on hover)
- Added image zoom effect on hover for visual engagement

1.2 STRUCTURE IMPROVEMENTS
-------------------------
- All cards now use flexbox for consistent internal layout
- Added card-link class for consistent "Read More" buttons
- Proper spacing with padding (24px body padding)
- Added meta information area for dates and categories

1.3 HOVER EFFECTS
----------------
- Subtle lift effect (translateY: -8px) on hover
- Shadow enhancement for depth
- Image zoom (scale 1.05-1.08) for visual feedback
- Smooth transitions (0.35s cubic-bezier) for professional feel
- Border color highlight on hover for department cards

1.4 CARD TYPES STANDARDIZED
---------------------------
- .card - Base card component
- .news-card - News articles with date badge
- .dept-card - Department cards with icon
- .faculty-card - Faculty member cards
- .stat-card - Statistics display cards

### Why These Changes Improve UX:

- Visual hierarchy: Cards now clearly separate content
- Scannability: Clear titles and consistent layout
- Interactive feedback: Hover states indicate clickable elements
- Professional appearance: Modern rounded corners and shadows
- Mobile-friendly: Responsive grid layouts
